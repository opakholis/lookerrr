# Lookerrr

this repo contains the completion of the final follow-up project from [sanbercode](https://sanbercode.com)


## Run Locally

Clone the project

```bash
  git clone git@github.com:opakholis/lookerrr.git
```

Go to the project directory

```bash
  cd lookerrr
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
